# tubes-oop-java-kelompok2
Aplikasi Kalkulator Keuangan Pribadi adalah aplikasi berbasis Java (Swing GUI) untuk mencatat dan menghitung pemasukan serta pengeluaran bulanan. Aplikasi ini menampilkan saldo akhir, ringkasan transaksi, dan visualisasi sederhana, serta menerapkan konsep OOP seperti enum, array, perulangan, event handling, dan exception handling.
